# Aurora Fin — лендинг финансового приложения

Сайт-лендинг для проекта Aurora Fin — умного приложения для учета финансов, планирования бюджета и аналитики.

## 📋 Описание

Одностраничный сайт (лендинг), который представляет концепцию финансового приложения Aurora Fin. Сайт содержит:
- Описание основных функций
- Блок "О нас"
- Контактную информацию
- Страницу политики конфиденциальности

## 🛠 Технологии

- HTML5
- CSS3 (адаптивная верстка)
